# RealTimeMovement

https://github.com/5tarlight/RealTimeMovement/assets/45203447/9d748927-566f-4788-8405-8b2d58ca9afc

## Features
- Join (random color)
- Move (with arrow keys)
- Quit

## Specs
- Spring Boot (Kotlin)
- HTML + JS
- Browser innate WebSocket
